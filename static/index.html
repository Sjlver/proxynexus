Proxy Nexus

Print Netrunner Proxy cards. Enter a card list, a NetrunnerDB link or select a set.

Only tested with Chrome.
In order to fit 9 images per page, select more settings, set Margins to Custom, and drag the top and bottom margins to 0.25". Refer to this example

Enter the URL of a published decklist from Netrunner DB.

Enter the URL of an unpublished decklist from Netrunner DB, from a user who has "Share your decks" enabled.

Select a set from the dropdown

Playsets are three copies of each card.

Sets that have been already released as PDF are ommited. To print the Magnum Opus cards, you can download the official PDFs from here:

Based on https://codepen.io/Theavon/pen/YQmgBG and https://gparmer.github.io/anrproxy/

Uses scans from "Android: Netrunner Cards up to Kitara @600dpi" Special thanks to /u/LepcisMagna/. Nisei cards are also included and unchanged from the print-and-play PDFs.

The scan have been despeckled using batch-despeckle.scm and enhanced using the tool mogrify with options "-modulate 105,105" from the Image Magick Suite. Here's a comparison of the improvements.
Check out the reddit post for more info!

Uses the NetrunnerDB API to query card IDs from names and deck URLs.
